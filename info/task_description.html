This series of tasks will help you learn the basics of OOP in Python, such as creating classes, attributes, and methods, inheritance, method overrides, using super(), and polymorphism.

Mission 6: Overriding methods.

  • 6.1 Add a start_engine method to the ElectricCar class that displays another message "The electric motor is started." and changes the property of working_engine to "Yes".
  • 6.2 Add a stop_engine method to the ElectricCar class that displays another message "The electric motor is stopped." and changes the property of working_engine to "No".
  • 6.3 Call the start_engine method for the my_electric_car object.
  • 6.4 Create my_electric_car2, start and stop its engine.
